# pyDFT

Educational Density Functional Theory (DFT) code for simple atomic systems, with:
- A standalone backend solver (CLI + Python API + FastAPI API).
- A pywebview desktop frontend that consumes the backend API.
- Unit and integration tests using `pytest`.
- Benchmark checks against reference values.

- `docs/user_manual.md`
- `docs/developer_manual.md`
